----------------------------------- 富民账户体系 ----------------------------------

富民II类户资金转入
     1.需要修改配置(网关)：
         acooly.openapi.client.fbank.payolGatewayUrl=http://127.0.0.1:8999/mock/gateway

     2.mock银行返回状态说明 (金额单位 为 分)
               交易金额 为 ：  失   败 ： 0 < amount <= 800
                              成   功 ： 800 < amount <= 19600
                              处 理 中： 19600 < amount

     3.异步查询，当金额为 19816 时，异步处理失败(金额单位 为 分)
